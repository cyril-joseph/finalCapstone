## 2. Conversion

Conversion formulae:
● Celsius from:
○ Fahrenheit: C = (F -32) x 5/9
○ Kelvin: C = K - 273.15
● Fahrenheit from:
○ Celsius: F = C x 9/5 + 32
○ Kelvin: F = k x 9/5 - 459.67
● Kelvin from:
○ Celsius: K = C + 273.15
○ Fahrenheit: K = (F + 459.67) x 5/9
